Data analysis for SDG score and CO2 emission¶

Background¶

The SDG index/score is a good indictor to understand countries sustainable efforts. However, the index reflects current status, and might not consider the future status.

In this Data Science session, we will compare the current SDG13 score (Climent action) and the forecest of CO2 Emissions.

Data sources¶

  • https://sdgindex.org/
  • https://data.worldbank.org/indicator/EN.ATM.CO2E.PC

Comparison SDG index vs CO2 Emission forecast¶

Compared Factors¶

  • SDG global index score
  • SDG13 score (Climate action)
  • CO2 emission forecast for 100 years

Observation¶

  • SDG13 score might not fully reflect the trend of CO2 Emission.
    • U.K. vs Japan - U.K. SDG13 score is worser than Japan one. However, the output of the forecast seems inverse.
  • The forecast seems simply considering only the trend. CO2 Emission after 100 years will be minus, but it is not correct.

Next steps¶

  • Investigate methods to calculate SDG13 score
  • Create the better forecast model for CO2 Emission
  • Check correlation between the trend of forecast and SDG13 score
  • Try to create new SDG13 index considering the forecast

def draw_html_with_fig(text, fig, col_width=300):
    figdata = BytesIO()
    fig.savefig(figdata, format='png')
    image_tag = f'<img src="data:image/png;base64,{b64encode(figdata.getvalue()).decode()}"/>'
    datatable = f'<table><tr><td width={col_width}>{text}</td><td>{image_tag}</td></tr></table>'
    display(HTML(datatable)) 

def run_prophet(e):
    t_prophet = pd.DataFrame()
    t_prophet['ds'] = e.index
    t_prophet['y'] = e['emissions'].values

    m = Prophet(weekly_seasonality=False, daily_seasonality=False)
    m.fit(t_prophet)

    future = m.make_future_dataframe(freq='y', periods=100)
    forecast = m.predict(future)

    return m, forecast

for df in dfs:

def add_emission(row, raw_e_country):
    index = str(row['date'].year)
    value = raw_e_country.loc[index]
    return value

def create_df_per_country(raw_e, country_code):
    raw_e_country = raw_e[raw_e['Country Code'] == country_code].loc[:,'1960':'2014']
    raw_e_country = raw_e_country.T
    raw_e_country.columns = ['value']

    date_range = pd.date_range(start='1960-12-31', end='2014-12-31', freq='y')
    e = pd.DataFrame(date_range, columns=['date'])
    e['emissions'] = e.apply(lambda row: add_emission(row, raw_e_country), axis=1)
    e.set_index('date', inplace=True)
    return e
